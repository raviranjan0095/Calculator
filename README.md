# calculator

The calculator program provides basic and scientific calculator functionality.

# Features
- Basic calculator: Basic calculator perform addition, subtraction, multiplication, and division operations.
- Scientific calculator: Scientific calculator perform sin, cos and tan trigonometric functions.

# Getting started
### Prerequisites:
- python3

- Run 'python calculator.py' to start the calculator.
- User choose the desired calculator type (Basic calculator, Scientific calculator). Options 1, 2 are valid.
- Based on calculator type user choose the desired operation.
